# Sistema de Control de Accesos RFID

Este repositorio contiene la documentación del **Sistema de Control de Accesos Empresarial basado en tecnología RFID**, desarrollado como proyecto académico para la asignatura de **Ingeniería de Software**.

## Descripción

El sistema permite gestionar y controlar el acceso de usuarios a instalaciones mediante **tarjetas RFID**, registrando los eventos de entrada y salida en una base de datos y permitiendo su consulta desde un panel administrativo.

## Funcionalidades principales

- Registro de usuarios  
- Asociación de tarjetas RFID  
- Validación de accesos  
- Registro de eventos de acceso  
- Generación de reportes  
- Exportación de reportes  

## Tecnologías utilizadas

- **Backend:** Node.js  
- **Frontend:** React  
- **Base de datos:** PostgreSQL  
- **Hardware:** Lector RFID  
- **Servidor:** Linux Ubuntu Server
